c++代码风格规范

为了提高协作效率，避免混乱提高代码的质量和可读性，我们作出以下代码风格约定， 使用google开源项目风格指南 一个好的编写代码的习惯，是成为优秀工程师的第一步。

在阅读此部分内容时你应该已经开始C++的学习或者已经完成了C++的学习

备注

由于b站等网站对于C++编程的教学十分详尽，本文档不会给出C++的教学，请读者自行搜索，也可点击传送门 b站黑马程序员。

声明

本项目并非原创内容，而是对由国内程序员凭热情创建和维护的 zh-google-styleguide 的迁移。
