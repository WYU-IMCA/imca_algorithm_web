Ubuntu (22.04版本)

安装虚拟机

ubuntu系统属于linux系统的一种发行版本，是与windows系统平级的一种系统，所以通常情况下在我们使用的电脑(安装windows系统)内使用ubuntu系统需要借助虚拟机来装载系统。 当然你也可以使用 双系统实体安装 。

访问 VMware官网。 选择对应系统的版本，例如在windows系统上使用虚拟机需要下载windows 版本。安装跟着提示走。

备注

注意最好不要下载安装到C盘，不要有中文路径。许可密钥可以在网上搜到，或者寻找破解版下载也可以。

安装Ubuntu镜像系统

在 Ubuntu官网 下载22.04版本镜像。

  • 打开VMware虚拟机，点击创建新的虚拟机

  • 选择典型，然后点击下一步

  • 勾选安装程序光盘映射文件，点击浏览选择之前下载好的镜像文件

  • 设置虚拟机名称和位置（不要在C盘），点击下一步

  • 分配磁盘大小，最好大于20G

  • 完成启动虚拟机

  • 点击下载

  • 后面跟着提示走即可

至此你已经完成了ubuntu系统的安装

备注

之后的教程如果没有特别说明默认在linux系统之下
